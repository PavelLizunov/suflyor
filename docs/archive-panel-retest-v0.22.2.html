suflyor — оверлей: узкая плашка («бар») вверху экрана. Это повторная проверка после исправлений по твоему прошлому отчёту. Установи сборку v0.22.2 и проверь каждый блок.

САМОЕ ВАЖНОЕ (F1): точность плеера чинилась только для НОВЫХ записей. Старые созвоны (записанные до этой сборки) останутся приблизительными — это нормально. Поэтому для блока F1 сделай новый созвон с записью (хотя бы 1–2 минуты речи) и проверяй на нём. Допустимо смещение в доли секунды; если звук уезжает на секунды или на следующую реплику — это БАГ.

Как открыть нужные окна

  1. F7 на баре → окно «Архив созвонов» (список; одна строка = один созвон).
  2. В строке справа иконки: ✎ переименовать · 🤖 авто-имя · 🗑 удалить · 💬 (облачко) открыть стенограмму · ↻ Summary пересоздать резюме.
  3. Клик по 💬 → окно «Стенограмма» (реплики + плеер внизу).
  4. Шестерёнка на баре → «Настройки» → вкладка «Память» (для блока F8).

Какой созвон брать

  • F1 (плеер): новый созвон с записью (сделай специально).
  • F3 (пересоздание): созвон, у которого уже есть резюме (сначала сделай резюме через ↻, потом жми ↻ ещё раз).
  • F4 (нет данных): самый пустой/старый созвон (без записи и без текста).
  • F6: нужен созвон с текстом, но без записи (где транскрипт есть, а аудио нет).
  • F2 (удаление): только НЕНУЖНЫЙ созвон — стирается безвозвратно!

Как отмечать

По каждому пункту: OK · БАГ · ? (непонятно/не нашёл) · скип. В поле — что именно не так (на БАГ — приложи скриншот отдельно). В конце нажми «Собрать отчёт» → «Копировать» или «Скачать .md» и верни мне.
