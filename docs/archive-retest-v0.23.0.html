📖 Инструкция — что нового и как проверять

Что изменилось: раньше у созвона со сводкой клик по кнопке сводки сразу предлагал «Пересоздать» (тратил платный AI-запрос и мог затереть хорошую сводку). Теперь у такого созвона основное действие — «Просмотреть»: открывает уже готовую сводку только для чтения, без обращения к ИИ. Пересоздание осталось, но как вторичная кнопка ↻ рядом — с подтверждением.

Где это (окно «Архив», F7)

В строке каждого созвона справа — действие сводки. Теперь оно в трёх видах:

Три состояния кнопки сводки в строке архива
сводка ЕСТЬСозвон с командой… 📄 Просмотреть↻
сводки НЕТСвежий созвон… ↻ Сформировать
нет данныхПустой созвон… Недостаточно данных

«Просмотреть» = открыть готовую сводку (без ИИ) · ↻ рядом = пересоздать (спросит) · «Сформировать» = сделать впервые

Какой созвон брать

  • Раздел A/B: созвон, по которому сводка уже делалась (есть готовая сводка).
  • Раздел C: созвон без сводки (свежий или тот, где не запускал сводку).
  • Раздел D: самый пустой созвон (без записи и без текста).
Раздел F — это диагностика бага «Память», а не проверка фикса. Баг ещё НЕ чинил — мне нужен один твой ответ, чтобы починить точно. Главное — пункт F-few (сократи списки памяти и проверь, остаётся ли чёрный экран).
