Суфлёр — ретест r4: ⭐-мультивыбор в стенограмме

Стенограмма: ⭐ теперь работает как в тайлах — накапливай несколько → одна запись. Открой F7 → 📄. Внизу «Экспорт».

Что изменилось

  • ⭐ у строки стенограммы — теперь переключатель-мультивыбор (как в тайлах), а не одиночный захват. Отметь несколько → одна кнопка «В память (N)» → одна склеенная запись.
  • Убрал прежнюю кнопку «Выбранное в память» из строки «выделить все» (её заменил ⭐-мультивыбор). Чекбоксы слева остались только для «Копировать выбранное».
⭐-мультивыбор → одна запись
Кликни ⭐ на нескольких строках. Звёзды должны подсветиться (акцент). Внизу — «В память (N)». Нажми.
Ожидается: звёзды накапливаются (N растёт); Настройки → Память → ОДНА склеенная запись из отмеченных строк.
Снять звезду (toggle)
Кликни отмеченную ⭐ ещё раз.
Ожидается: снимается, N уменьшается.
Одна ⭐ — с правкой
Отметь ОДНУ строку ⭐ → внизу поле с текстом → поправь → «В память».
Ожидается: одна запись (с твоей правкой).
Выделение мышью → в память (не сломалось)
Выдели часть текста реплики мышью → правый клик → «Выделенное в память» → редактор → сохрани.
Ожидается: работает; при этом ⭐-отметки сбрасываются (общий редактор).
Копирование чекбоксами (регрессия)
Отметь чекбоксы слева у пары строк → «Копировать выбранное» → вставь куда-нибудь.
Ожидается: копирование в буфер работает как раньше (чекбоксы ≠ звёзды, независимы).
