Суфлёр — ретест r3: раскладки + склейка памяти

Сборка от 2026-07-03 (r3). Запусти обычным ярлыком. Заполни → внизу «Экспорт результатов».

Что нового в r3

  • Ctrl+C / V / X / A под ЛЮБОЙ раскладкой — не только русской. Теперь ловим физическую клавишу (не символ), так что работает и на др. языках. Главное — проверить под РУС во всех полях.
  • Склейка памяти — несколько ⭐ (или чекбоксов в стенограмме) → ОДНА запись, не N разрозненных (фикс z14-backup → 3 строки).

Как тестировать G1

  • Переключи раскладку на РУССКУЮ и жми Ctrl+C/V/X/A физическими клавишами (C/V/X/A), не думая о букве под ними.
  • Старые шимы я оставил временно (подстраховка) — удалю, когда подтвердишь, что новый механизм работает.

G1. Ctrl+C/V/X/A под русской раскладкой

Настройки → «Свой факт» (тот самый баг)
РУС раскладка. Настройки → Память → поле «Свой факт» (или любое поле ввода). Скопируй что-то в другом месте → в поле Ctrl+V.
Ожидается: вставляется. Также проверь Ctrl+A (выделить всё в поле), Ctrl+X (вырезать).
Окно «Написать» (✏)
РУС. Открой ✏ «Написать ассистенту». Печатай, Ctrl+A, Ctrl+C, Ctrl+V, Ctrl+X.
Ожидается: все четыре работают.
Редактор факта + «Продолжить диалог»
РУС. В тайле: выдели текст → правый клик → «Выделенное в память» → в поле снизу Ctrl+V. Также в «Продолжить диалог» Ctrl+V.
Ожидается: вставка работает в обоих.
Палитра F4
РУС. F4 → в поле поиска Ctrl+V / печать / Ctrl+A.
Ожидается: работает.
Копирование с выделяемого текста ответа
РУС. Выдели текст в тайле/сводке → Ctrl+C (и Ctrl+A) → вставь куда-нибудь.
Ожидается: копирует выделенное / всё.
Латинская раскладка не сломалась (регрессия)
Переключись на EN. Те же Ctrl+C/V/X/A в любом поле.
Ожидается: работают как раньше (это важно — не должно сломаться).
Другая раскладка (если есть)
Если есть 3-я раскладка (укр/нем/фр) — Ctrl+V в любом поле.
Ожидается: работает. Нет 3-й раскладки → «Пропустить».

G2. Склейка памяти (одна запись, не разрозненно)

Несколько ⭐ на блоках → ОДНА запись
В тайле/сводке отметь ⭐ несколько блоков (напр. имя / подсеть / IP) → «В память (N)». Открой Настройки → Память.
Ожидается: ОДНА запись (склейка через «; »), а не N отдельных строк. Раньше дробилось.
Одиночная ⭐ по-прежнему с правкой
Отметь ОДИН блок ⭐ → поле правки снизу → поправь → «В память».
Ожидается: как раньше (трим перед сохранением).
Стенограмма: чекбоксы → ОДНА запись
Архив F7 → 📄 стенограмма. Отметь чекбоксы нескольких строк → кнопка «Выбранное в память» (рядом с «Копировать выбранное»). Проверь Настройки → Память.
Ожидается: одна склеенная запись из отмеченных строк.

D. Старое не сломалось

Регрессия
Быстро: выделение текста (поблочно + режим «Выделить текст»), наслоения нет, скролл/драг тайла, звёзды приглушённые.
Ожидается: всё как в r2.
