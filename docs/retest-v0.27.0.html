Суфлёр v0.27.0 — приёмка релиза

Финальная проверка на УСТАНОВЛЕННОМ билде (после установки setup.exe). Всё это уже проверено по r1–r4 — тут короткое подтверждение на реальном инсталляторе.

Что в релизе

  • Выделение текста мышью в тайлах / сводках / архиве / стенограмме + правый клик «Копировать / В память»; фикс наслоения блоков; режим «Выделить текст» (сквозное); приглушённые звёзды.
  • Ctrl+C/V/X/A под любой раскладкой (не только RU).
  • Связная память: несколько ⭐ → ОДНА запись (тайлы и стенограмма).
  • Внутри: схема памяти v2 (спящая, под будущую нормализацию фактов) — на поведение не влияет.
Выделение + правый клик
Тайл/сводка/архив/стенограмма: выдели текст → правый клик → «Копировать выделенное» / «Выделенное в память». Работает.
Нет наслоения
Заголовки/код/длинные абзацы в сводке не налезают друг на друга (обе темы).
Режим «Выделить текст»
«⋯» → «Выделить текст» → тащишь через блоки → копия/в память; «Обычный вид» назад.
Ctrl+C/V/X/A под РУС
Русская раскладка: Ctrl+V в «Свой факт» / ✏ / «Продолжить диалог» / F4 вставляет; Ctrl+C/A/X работают. EN не сломан.
Склейка памяти — тайлы
Несколько ⭐ на блоках → «В память (N)» → в Настройки→Память ОДНА запись.
Склейка памяти — стенограмма
Несколько ⭐ на строках → «В память (N)» → одна запись. Чекбоксы слева → «Копировать выбранное» (буфер) работают.
Старое не сломалось
Запуск, бар на месте, тайлы/сводки/архив/стенограмма открываются, скролл/драг, 📋-копия, «Продолжить диалог», озвучка — как раньше.
