# suflyor v0.23.0

Минорный релиз: большая панель **«Архив»** + доработки по live-отчётам тестеров.

## ✨ Новое

### Панель «Архив»
По каждой прошедшей сессии теперь доступны:
- полная **стенограмма** созвона;
- **аудиоплеер** записи;
- **ретроспективная сводка** (можно сформировать и пересоздать);
- **удаление** сессии.

Доработки панели (фиксы F1–F8): точный seek плеера (сохранение позиции в БД + миграция), фикс краша при удалении, подтверждение перед пересозданием сводки, кнопка «↻» неактивна у пустых записей, SVG-иконки вместо эмодзи, устранение тормозов/перерасхода памяти на вкладке «Память» при больших списках.

### Просмотр готовой сводки
У строки с уже готовой сводкой основное действие — **«Просмотреть»**: сводка открывается только для чтения (markdown-тайл, как ответ ИИ), **без повторного прогона модели**.

## 🐞 Исправления (по live-отчёту тестера)
- **Авто-переключение строки.** После того как сводка сформирована, строка архива сразу становится «Просмотреть» — больше не нужно закрывать и открывать архив.
- **«Пересоздать» реально пересчитывает.** Раньше при неизменной стенограмме возвращался прежний (закэшированный) текст; теперь это полноценный новый прогон. При сбое пересоздания **старая сводка не теряется**.
- **Пустые созвоны.** Сессия без записи и без текста показывает «Недостаточно данных» вместо неактивного «Сформировать».

## 🩺 Диагностика
- В «Копировать отчёт» добавлены **сведения о системе** — ОС и видеокарта (строка `System: … · GPU: …`). Помогает быстрее локализовать проблемы отображения.
- Копируемый отчёт **больше не раскрывает имя пользователя ОС**: путь к локальным моделям маскируется как `%USERPROFILE%`. (Отчёт по-прежнему безопасно вставлять в переписку — адреса/хосты и раньше скрывались.)
