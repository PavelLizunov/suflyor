🎧 Плеер стенограммы (ваш запрос)

P1. Мини-плеер отображается
Откройте стенограмму сессии с аудио. Внизу — строка плеера: play/pause · полоса перемотки · время · кнопка скорости · кнопка громкости 🔊.
НОВОЕP2. Скорость: 1× → 1.5× → 2×
Тап по кнопке скорости циклит 1× → 1.5× → 2× → 1×. Во время воспроизведения звук идёт быстрее, а полоса перемотки и время НЕ рассинхронизируются (не убегают вперёд/назад относительно того, что слышно).
НОВОЕP3. Громкость (усиление): 1× → 2× → 3×
Тап по кнопке 🔊 циклит 1× → 2× → 3× → 1×. Тихая запись становится громче. На 3× возможны искажения только у изначально громких записей — это ожидаемо.
P4. Скорость/громкость можно выставить ДО play
Не запуская воспроизведение, выставьте, скажем, 2× и 🔊 2×, затем нажмите play — должно заиграть сразу на выбранных значениях.
P5. Сброс при смене сессии
Выставьте не-1× значения, закройте стенограмму, откройте другую сессию — скорость и громкость снова 1× / 1×.

🗣 Диаризация «По голосам» + фиксы ревью

V-1D1. Кнопка установки моделей
Если модели диаризации НЕ установлены, а сессия пригодна (завершена, есть системное аудио) — показывается подпись «Для определения говорящих нужно один раз скачать модели (~30 МБ)» и кнопка «Установить модели говорящих». Клик → скачивание → после успеха появляются кнопки определения.
⚠ У вас модели уже стоят → сразу видны кнопки определения (это тоже «ок»). Кнопку установки можно проверить, только удалив модели.
D2. Запуск определения говорящих
Степпер «Собеседников» (по умолчанию 2) + кнопка «Определить говорящих» → идёт анализ → переключается на вид «По голосам» с цветными метками говорящих.
V-2D3. Подпись «Собеседников (без вас):»
Подпись у степпера читается «Собеседников (без вас):» — счётчик считает удалённых собеседников, не вас (диаризация идёт по системному аудио, ваш микрофон — отдельный канал).
V-3D4. Переименование 5+ говорящих не вылезает за окно
Если говорящих 5 и больше, строка переименования (цветная точка + поле имени на каждого) прокручивается по горизонтали внутри окна, а не расползается за его края и не прячет кнопку. Для 2–3 говорящих полосы прокрутки не видно (умещается).
⚠ Проверяется, только если задать 5+ собеседников и запустить определение.
I-1D5. Баннер о неточных таймингах для старых записей
На записи, сделанной старым билдом (все ваши текущие записи — старые), над кнопками показывается предупреждение: «Тайминги говорящих для этой записи могут быть неточными.» Определение всё равно доступно (баннер не блокирует).
I-5D6. Понятные сообщения об ошибке
Если определение не удаётся: сообщение конкретное — например «Речь не распознана — говорящие не найдены.» или «Запись длиннее 3 часов…» — а не общий «Не удалось…». В тексте НЕ должно быть путей к файлам (C:\...).
⚠ Провал трудно вызвать намеренно; если не удаётся — ⏭.
D7. Переименование говорящего сохраняется
В виде «По голосам» впишите имя говорящему, Enter — стенограмма перекрашивает/переименовывает его реплики; имя сохраняется после переоткрытия.
D8. Переключение «По ролям» ↔ «По голосам»
Тумблер «По ролям» / «По голосам» переключает вид (роли Микрофон/Система ↔ per-speaker) без ошибок.
