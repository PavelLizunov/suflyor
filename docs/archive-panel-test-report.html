suflyor — это оверлей: узкая плашка («бар») вверху экрана. Проверяем новые функции архива созвонов. Приложение должно быть уже запущено.

Как открыть нужные окна

  1. Найди бар — узкую плашку вверху экрана.
  2. Нажми F7 → откроется окно «Архив созвонов» (список прошлых созвонов; одна строка = один созвон).
  3. В строке созвона справа есть маленькие иконки. Новые — 🗑 (удалить) и значок-облачко (наведи мышь — подсказка «View transcript»).
  4. Клик по облачку → откроется окно «Стенограмма» с репликами созвона.
  5. Плеер — это нижняя полоска окна «Стенограмма» (кнопка ▶ + полоса перемотки + время). Он есть только у созвонов с записью; если записи нет — там надпись «Аудио не сохранено».

«Клик по строке» — это про окно «Стенограмма»

Клик по реплике в списке стенограммы (НЕ по созвону в архиве) → звук пойдёт с момента этой реплики.

Какой созвон брать для каждого раздела

  • Раздел 1 (плеер): созвон с записью — бери посвежее (у старых запись могла очиститься).
  • Пункт 2.4 (пустой транскрипт): наоборот, старый созвон без текста.
  • Раздел 3 (удаление): только на НЕНУЖНОМ созвоне — стирается безвозвратно!

Как отмечать

По каждому пункту жми: OK (работает) · БАГ (не так) · ? (непонятно/не нашёл) · скип (не проверял). В поле — что именно не так (на БАГ — приложи скриншот отдельно). В конце нажми «Собрать отчёт → копировать» и верни текст.
