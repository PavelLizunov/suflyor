Суфлёр v0.28.0 — приёмка

Проверка на УСТАНОВЛЕННОМ билде (сначала запусти setup.exe, потом кликай ярлык — не target/release). Версия рабочая, при релизе номер могу сменить. Отметь каждый пункт ✅/❌/⏭ и жми «Экспорт результатов».

Что изменилось

  • Память → надёжная нормализация. ⭐-факт теперь чистится ИИ в ДОСЛОВНУЮ цитату (не выдумывает и не склеивает слова). Если ИИ был выключен — факт не теряется («ждёт ИИ») и до-чищается сам, когда ИИ вернётся.
  • Стенограмма. Звёзды больше не дерутся со скроллом; тайминги идут по порядку и кликаются до конца.
  • Тайлы: выделение звёздами. SHIFT-клик по ⭐ выделяет диапазон блоков; на панели «В память» появилась кнопка «Копировать». Старый режим «Выделить текст» (ломал вёрстку на большом тексте) удалён.

Память — нормализация (нужен ВКЛючённый ИИ: локальный сервер или облако)

Дословная чистка
Скажи кривую фразу (со словами-паразитами) и ⭐-ни строку стенограммы ЛИБО ⭐-ни блок тайла. В Настройки → Память факт стал коротким и чистым, и это ДОСЛОВНЫЙ кусок сказанного — без выдуманных слов. Бейдж «подчищено ИИ · было: <сырьё>».
Ничего не выдумано / не склеено
Проверь на паре фактов: в сохранённом тексте нет слов, которых ты не говорил, и нет склейки далёких кусков в ложное утверждение (напр. не должно превратить «сервер Бета — тестовый … продакшн — Альфа» в «Бета продакшн»). Всегда непрерывная цитата.
ИИ выключен → факт не теряется
Выключи ИИ (локальный сервер не запущен / облако недоступно). ⭐-ни факт → он сохраняется с бейджем «ждёт ИИ» (НЕ пропадает, НЕ помечается «почищено»).
ИИ вернулся → до-чистка сама
Включи ИИ обратно. Открой Настройки → Память (или перезапусти прогу и подожди ~15 c) → строки «ждёт ИИ» сами становятся «подчищено ИИ». Ручных действий не требуется.
Правка факта убирает бейдж
В Настройки → Память отредактируй текст факта → сохрани. Бейдж («подчищено ИИ» / «ждёт ИИ») пропадает — теперь это твой текст, не ИИ.
Кривой конфиг не залипает
(если легко проверить) При неверной модели/ключе ИИ факт НЕ висит вечно в «ждёт ИИ» — становится «почищено» (эвристика). Не блокирует остальные.

Стенограмма

Звёзды не дерутся со скроллом
Открой длинную стенограмму, прокрути. ⭐ справа не наезжают на полосу прокрутки, каждая ⭐ кликается.
Тайминги по порядку
В стенограмме (система + микрофон вперемешку) тайминги идут по возрастанию ДО конца — внизу нет перепутанных/обратных времён. Клик по поздней строке перематывает верно.

Тайлы — выделение звёздами (⭐)

SHIFT-клик = диапазон
В тайле кликни одну ⭐, затем SHIFT+клик по ⭐ на несколько блоков ниже → выделяется весь диапазон подряд; «В память (N)» показывает число блоков в диапазоне.
«Копировать» на панели
С выделенными блоками нажми «Копировать» → вставь куда-нибудь: в буфере текст этих блоков. Выделение (⭐) при этом НЕ сбрасывается.
Копия = правленый текст (1 блок)
Выдели РОВНО один блок, отредактируй текст в поле снизу, нажми «Копировать» → в буфере ТВОЙ правленый текст (то же, что сохранила бы «В память»).
Большой текст не ломает тайл + режима «Выделить текст» больше нет
Длинный/большой ответ больше не ломает вёрстку тайла и не выкидывает кнопки за край. В меню «⋯» пункта «Выделить текст» / «Обычный вид» больше НЕТ.

Ничего не сломалось

Регрессия
Запуск, бар на основном мониторе, тайлы/сводки/архив/стенограмма открываются, скролл/драг, 📋-копия, «Продолжить диалог», озвучка. Одиночная ⭐ + правый клик «Выделенное в память» на блоке — как раньше.
